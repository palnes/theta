# Border Tokens

Intent-based border tokens that adapt to themes and express design decisions. All border tokens are semantic - there are no primitive border tokens in this design system.

export const BorderVisualization = () => {
  const { data, loading, error } = useDocumentationData();
  
  if (loading) return <div>Loading borders...</div>;
  if (error) return <div>Error loading borders: {error}</div>;
  if (!data) return null;
  
  const borders = getSemanticBordersFromDocs(data.sys?.border || []);
  
  return (
    <>
      <h2>Border Widths</h2>
      <div style={{ marginBottom: 48 }}>
        <h3>Semantic Width Scale</h3>
        <div style={{ display: 'grid', gridTemplateColumns: 'repeat(2, 1fr)', gap: 16 }}>
          {borders.width.map(({ key, value, description, cssVariable, jsPath }) => (
            <div key={key} style={{ 
              padding: 24, 
              background: 'var(--sys-color-surface-secondary)', 
              borderRadius: 'var(--sys-radius-large)',
              border: `1px solid var(--sys-color-border-subtle)`
            }}>
              <div style={{ display: 'flex', alignItems: 'center', gap: 16 }}>
                <div style={{ 
                  width: 80, 
                  height: 80, 
                  background: 'var(--sys-color-surface-raised-default)', 
                  borderRadius: 'var(--sys-radius-medium)',
                  border: `var(--sys-border-width-${key}) solid var(--sys-color-border-strong)`
                }} />
                <div>
                  <h4 style={{ margin: '0 0 8px 0', textTransform: 'capitalize' }}>{key}</h4>
                  <code style={{ fontSize: 13, color: 'var(--sys-color-text-secondary)' }}>
                    {cssVariable}
                  </code>
                  <code style={{ display: 'block', fontSize: 12, color: 'var(--sys-color-text-tertiary)', marginTop: 2 }}>
                    {jsPath.replace('tokens.', '')}
                  </code>
                  <div style={{ fontSize: 12, color: 'var(--sys-color-text-muted)', marginTop: 8 }}>
                    {description}
                  </div>
                </div>
              </div>
            </div>
          ))}
        </div>
      </div>
      
      <h2>Border Styles</h2>
      <div style={{ marginBottom: 48 }}>
        <h3>Semantic Style Options</h3>

        </div>

    </>
  );
};

## Border Guidelines

- Use semantic naming that expresses intent, not visual properties
- Ensure sufficient contrast with border colors for accessibility
- Limited scale options maintain visual harmony across themes

## Related Tokens

- **Border Colors**: See [Semantic Colors](/?path=/docs/token-reference-semantic-colors--docs) for `sys.color.border.*`
- **Border Radius**: See [Semantic Radius](/?path=/docs/token-reference-semantic-radius--docs) for corner rounding
