# Color Calculator

Convert between standard RGB color formats (hex, rgb, rgba) and the DTCG color token format.

The Design Tokens Community Group (DTCG) specification uses a normalized color format where RGB components are expressed as decimal values between 0 and 1, rather than the traditional 0-255 range.
