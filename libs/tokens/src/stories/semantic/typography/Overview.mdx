# Typography System Overview

Our semantic typography system provides pre-composed text styles that ensure consistent hierarchy across the application.

## Typography Categories

### [Headings](/story/token-reference-semantic-typography-headings--docs)
Typography tokens for headings and titles. Use these for page titles, section headers, and other prominent text.

### [Body](/story/token-reference-semantic-typography-body--docs)
Typography tokens for body text and paragraphs. Use these for regular content and descriptions.

### [Action](/story/token-reference-semantic-typography-action--docs)
Typography tokens for interactive elements like buttons and links. These ensure consistent styling for clickable text.

### [Label](/story/token-reference-semantic-typography-label--docs)
Typography tokens for form labels and UI annotations. Use these for field labels, helper text, and small UI text.
