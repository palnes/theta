# Border Radius System

Consistent border radius values for rounded corners throughout the design system.

export const RadiusScale = () => {
  const { data, loading, error } = useDocumentationData();
  
  if (loading) return <div>Loading radius values...</div>;
  if (error) return <div>Error loading radius: {error}</div>;
  if (!data) return null;
  
  const radiusTokens = getSemanticRadiusFromDocs(data.sys?.radius || []);
  
  return (
    <div style={{ display: 'grid', gridTemplateColumns: 'repeat(auto-fill, minmax(100px, 1fr))', gap: 24, marginBottom: 48 }}>
      {radiusTokens.map(({ key, tokenValue, cssVariable }) => (
        <div key={key} style={{ textAlign: 'center' }}>
          <div
            style={{
              width: 80,
              height: 80,
              backgroundColor: 'var(--sys-color-action-primary-default)',
              border: '1px solid var(--sys-color-border-default)',
              borderRadius: `var(--sys-radius-${key})`,
              margin: '0 auto 12px',
            }}
          />
          <div style={{ fontSize: 12, fontWeight: 600 }}>{key.toUpperCase()}</div>
          <code style={{ fontSize: 10, color: 'var(--sys-color-text-muted)', display: 'block' }}>{cssVariable}</code>
          <div style={{ fontSize: 10, color: 'var(--sys-color-text-muted)' }}>{tokenValue}</div>
        </div>
      ))}
    </div>
  );
};

## Radius Guidelines

- **none (0px)**: Sharp corners, data tables, code blocks
- **sm (4px)**: Small interactive elements, inputs, checkboxes
- **md (8px)**: Buttons, dropdown menus, tooltips
- **lg/xl (12px)**: Cards, modals, large containers
- **full (9999px)**: Pills, badges, circular elements

Use consistent radius values to create visual cohesion across your interface.
