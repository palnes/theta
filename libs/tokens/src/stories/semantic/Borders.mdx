# Border Tokens

Border width and style tokens. All border tokens are semantic - there are no primitive border tokens.


<BorderDisplay />



## Border Guidelines

- Use semantic naming that expresses intent, not visual properties
- Ensure sufficient contrast with border colors for accessibility
- Limited scale options maintain visual harmony across themes

## Related Tokens

- **Border Colors**: See [Semantic Colors](/?path=/docs/token-reference-semantic-colors--docs) for `sys.color.border.*`
- **Border Radius**: See [Semantic Radius](/?path=/docs/token-reference-semantic-radius--docs) for corner rounding
- **Spacing**: Borders often work with spacing tokens for visual rhythm
