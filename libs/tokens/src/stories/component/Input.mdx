# Input Component Tokens

Complete specification of all input component tokens organized by category.

## Token Structure

Input tokens follow the pattern: `cmp.input.[category].[size].[property].[state]`

## Size Variants

Input supports three size variants:
- **Small**: Compact inputs
- **Medium**: Default input size
- **Large**: Larger inputs for emphasis

## Typography

Each size variant includes:
- Font size
- Line height for consistent text rendering
- Font weight for proper text emphasis
