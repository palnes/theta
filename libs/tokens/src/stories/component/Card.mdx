# Card Component Tokens

Complete specification of all card component tokens organized by category.

## Token Structure

Card tokens follow the pattern: `cmp.card.[category].[size/variant].[property].[state]`

## Size Variants

Card supports three padding sizes:
- **Small**: Compact cards
- **Medium**: Default card size
- **Large**: Spacious cards

## Shadow Variants

Card supports multiple shadow levels:
- **Base**: Default elevation
- **Large**: Higher elevation
- **Spread**: Dramatic spread shadow
