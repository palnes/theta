# Checkbox Component Tokens

Complete specification of all checkbox component tokens organized by category.

## Token Structure

Checkbox tokens follow the pattern: `cmp.checkbox.[category].[property].[state]`

## States

Checkbox supports multiple states:
- **Default**: Unchecked state
- **Hover**: Mouse hover interaction
- **Active**: During click/press
- **Selected**: Checked state
- **Disabled**: Non-interactive state
- **Error**: Validation error state
