# Badge Component Tokens

Complete specification of all badge component tokens organized by category.

## Token Structure

Badge tokens follow the pattern: `cmp.badge.[category].[size/variant].[property].[state]`

## Size Variants

Badge now supports three size variants:
- **Small**: Compact badges for tight spaces
- **Medium**: Default badge size
- **Large**: Emphasized badges for important information
