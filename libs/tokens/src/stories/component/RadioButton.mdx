# Radio Button Component Tokens

Complete specification of all radio button component tokens organized by category.

## Token Structure

Radio button tokens follow the pattern: `cmp.radioButton.[category].[property].[state]`

## Design Details

- **Border Radius**: Uses full radius for perfect circle appearance
- **Transition**: Smooth state changes with fast duration
- **States**: Supports default, hover, selected, disabled, and error states
