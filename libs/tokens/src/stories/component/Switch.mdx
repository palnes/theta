# Switch Component Tokens

Complete specification of all switch component tokens organized by category.

## Token Structure

Switch tokens follow the pattern: `cmp.switch.[category].[property].[state]`

## Animation

- **Transition Duration**: Uses normal duration for smooth thumb sliding
- **States**: Animated transitions between default, hover, active, and disabled states
